# Customer Churn Prediction using Machine Learning

## 📌 Overview
This project predicts whether a customer is likely to churn using machine learning models. It also provides actionable business recommendations and explainable insights.

## 🚀 Features
- Churn prediction using ML models
- Stacked ensemble model (SVM + Random Forest + Logistic Regression)
- Explainable AI using SHAP
- Decision system for business actions
- Real-time prediction for new customers

## 🧠 Technologies Used
- Python
- Scikit-learn
- Pandas, NumPy
- SHAP (Explainable AI)
- Streamlit (UI)

## 📊 Workflow
1. Data preprocessing
2. Feature engineering
3. Model training
4. Stacking ensemble
5. Prediction
6. Explainability (SHAP)
7. Business decision engine

## 📈 Model Architecture
- Base Models: SVM, Random Forest
- Meta Model: Logistic Regression
- Output: Churn Probability + Action + Reason

## 🎯 Output Example
Churn Probability: 0.81
Risk Level: High

Recommended Action:
→ Personalized Retention Offer

Reason:
→ High monthly charges
→ Low tenure
